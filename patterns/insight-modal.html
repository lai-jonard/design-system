Insight modal

Clicking an insight row fires la-insight-select; the consumer populates and opens la-insight-modal. The modal overlays the page — it does not navigate. The consumer controls all state; the component never routes itself.

Examples

Insight card — click to open modal
Event log — la-insight-select
Click the insight card above to see the event fire.

Event chain

click insight row → la-insight-select → open modal "View document" → la-insight-navigate → close + navigate upload action → la-insight-upload → processing → resolved X / backdrop → la-insight-close → close modal

Rules

  • The modal overlays the page — it does not navigate.
  • The consumer catches events and controls modal state; the component never routes itself.
  • Close on X, backdrop click, or Escape.
  • Populate all props before setting open = true — never show a partially populated modal.
