Inline metadata

Middle dot (·) separator for horizontal label runs in document headers, list rows, or cards. Separator is a discrete DOM node — lighter than surrounding text so it recedes and labels stand out.

Examples

Document header — entity, area, date

Acme Corp. · Employment · 3 Mar 2024

Document row — name, format, size

Consulting Agreement · PDF · 240 KB

Implementation — discrete DOM nodes for each separator
<span>Acme Corp.</span> <span class="sep"> · </span> <span>Employment</span> <span class="sep"> · </span> <span>3 Mar 2024</span>

Style .sep with color: var(--la-color-text-faint) and user-select: none.

Rules

  • Separator: middle dot U+00B7 (·) — never slash, pipe, dash, or vertical rule.
  • Colour: --la-color-text-faint — lighter than surrounding text so the dot recedes.
  • user-select: none on separator spans — excluded from copy/paste.
  • Never orphan a dot — only render a separator when the preceding item is present.
  • Never hardcode · inside a string literal — keep separators as discrete DOM nodes so conditional visibility is handled in the template.
