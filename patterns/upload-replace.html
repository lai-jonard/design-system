Upload / Replace

Adding a new document or replacing an existing one. Replace flows through a modal wrapping la-upload. Add surfaces inline on the legal area page — no modal.

Examples

Replace — modal trigger
Replace document
Cancel Upload
Add — inline upload surface (no modal)

Rules

  • Replace = modal wrapping la-upload. Add = inline on the page, no modal.
  • File size limit is 50 MB — never 10 MB in copy.
  • The Upload button in the footer is disabled until a file is selected.
  • On upload, status transitions to processing; existing content stays visible during processing.
  • Going dark or hiding content during processing would alarm users — content stays in view.
  • On failure, status transitions to flagged-ai with a retry CTA; the original document is unchanged.
