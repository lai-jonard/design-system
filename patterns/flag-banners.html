Flag banners

Persistent in-page alerts for flagged, processing, or failed document states. Use la-alert inline — not a toast, not a modal. Placed immediately below the page header, above main content.

Examples

Flagged by AI — warning variant
This document was flagged by our AI. Review the highlighted sections and re-upload a corrected version.
Processing — info variant
Analysing document… This usually takes less than a minute.
Upload failed — danger variant
Upload failed. The file may be corrupted or unsupported. Try again.

Rules

  • Use la-alert inline in the page — not a toast, not a modal.
  • Placed immediately below the page header, above main content.
  • Variants map to state: warning for flagged-AI, info for processing, danger for failed actions.
  • Dismiss only when the underlying state is resolved — not on user click unless the flag is purely informational.
  • Always include a short explanation or action — never just a status label.
