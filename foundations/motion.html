Foundations · Motion

Motion

Source of truth is motion.css. Utility classes + tokens handle every animation — never handwritten keyframes, never ease-in.

  • GPU-safe only — animate transform and opacity. Never height, width, top, left, background, or color.
  • Never scale from zero — minimum starting scale is 0.96.
  • Max 300ms for repeated interactions, 500ms for anything triggered often.
  • Stagger: 40–60ms between list items.
  • Reduced motion: motion.css automatically collapses all durations to 0ms when prefers-reduced-motion is set.

Durations

TokenValueUse
--dur-instant100msTooltips, feedback too small to read as motion.
--dur-fast150msHovers, press feedback.
--dur-base200msStandard in/out transitions.
--dur-modal250msModals, dialogs, overlays.
--dur-page350msPage or layout-level transitions.
Duplicate duration scale in tokens.ts. A legacy --la-dur-fast/base/modal/slow set still exists alongside --dur-*. Use --dur-* for new code; a cleanup pass on the legacy set is pending.
