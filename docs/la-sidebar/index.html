la-sidebar

The navigation sidebar. Placed in the sidebar slot of la-app-shell. Shows the logo, user identity, navigation areas, and lock state for gated areas.

Properties

Property Attribute Type Description
activeArea active-area string e.g. dashboard, corp-gov, employment. Drives the active styling.
activeSubtab active-subtab string e.g. company-manual, people. Active styling for sub-navigation.
unlockedAreas unlocked-areas Array JSON string array of unlocked areas, e.g. ["corp-gov", "employment"].
collapsed collapsed boolean If true, collapses to an icon-only mode.
userInitials user-initials string e.g. AC
userName user-name string e.g. Alex Chen
userRole user-role string e.g. Founder
logoSrc logo-src string Path or URL to the square logo.
wordmarkSrc wordmark-src string Path or URL to the wide wordmark.

Events

  • la-nav-change — detail { area, subtab }. Fired when a nav item or sub-tab is clicked. The event bubbles up to la-app-shell; listen there to update UI state.
