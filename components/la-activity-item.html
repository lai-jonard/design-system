la-activity-item

A single event row in a chronological activity feed. Renders an icon circle, a connecting vertical line to the item below, an event description slot, and a timestamp. Stack multiple items to form a timeline — the connector lines chain automatically. Set last on the final item to suppress the trailing line.

Anatomy

Two columns: the icon column on the left (circle + connector line), and the body on the right (event text + timestamp).

Anatomy — single item (last)

Anatomy regions
RegionSourceRole
Icon circle .icon-circle + slot="icon" 28 px circle housing a Phosphor icon. Background is --la-color-bg with a light border — intentionally neutral, no status colour.
Connector line .icon-col::after 1.5 px vertical line from the bottom of the icon circle to the top of the next item's circle. Hidden when last is set.
Event text Default slot Primary description of the event. May include inline <a> links to referenced documents. 16 px / normal weight.
Timestamp time prop → .time Relative or absolute time string. 14 px / faint colour. Optional — omit for system events with no meaningful time.

la-activity-item

A single event row in a chronological activity feed — icon circle, connector line, event text, and timestamp. Compose to form a timeline.

Properties

PropertyAttributeTypeDefaultDescription
time time string '' Timestamp string shown below the body (e.g. "2 hours ago")
last last boolean false Suppresses the connecting line. Set on the final item.

Slots

SlotDescription
(default) Event description text. May include inline `` links.
icon Icon inside the circle. Use a Phosphor `` tag.

CSS custom properties

No public CSS custom properties. All dimensions, colours, and spacing reference design tokens. The feed background (--la-color-bg-subtle) is the responsibility of the wrapping container, not the item.

Usage examples

Static HTML feed

<div class="activity-feed">
  <la-activity-item time="1 hour ago">
    <i slot="icon" class="ph-fill ph-eye"></i>
    Viewed <a href="../docs/cap-table">Cap Table (as of Jan 2026)</a>
  </la-activity-item>

  <la-activity-item time="13 Apr 2026">
    <i slot="icon" class="ph-fill ph-paper-plane-tilt"></i>
    DD pack sent — 11 documents shared
  </la-activity-item>

  <la-activity-item time="10 Apr 2026" last>
    <i slot="icon" class="ph-fill ph-envelope-simple"></i>
    Invitation accepted
  </la-activity-item>
</div>

Lit template from data array

// ActivityItemData: { icon: string; text: string; time: string }
${items.map((item, i) => html\`
  <la-activity-item
    .time=\${item.time}
    ?last=\${i === items.length - 1}
  >
    <i slot="icon" class="\${item.icon}"></i>
    \${unsafeHTML(item.text)}
  </la-activity-item>
\`)}

Recommended container styles

.activity-feed {
  background: var(--la-color-bg-subtle);
  border-radius: var(--la-radius-lg);
  padding: var(--la-space-md) var(--la-space-lg);
}

Notes

  • Always set last on the final item. The connector line uses a CSS ::after pseudo-element that extends to the bottom of the icon column — without last, it trails into empty space below the item.
  • Feed background is external. The component itself has no background. Wrap items in a .activity-feed container (or similar) with background: var(--la-color-bg-subtle) and appropriate padding.
  • Icon choice: Use filled Phosphor variants for event icons — ph-fill ph-eye, ph-fill ph-paper-plane-tilt, ph-fill ph-envelope-simple, ph-fill ph-prohibit. The filled weight reads better at 13 px inside the circle.
  • HTML in text: When rendering from a data array using Lit, use unsafeHTML(item.text) for text that may contain inline links. Only use this with developer-controlled data — never with user-generated content.
  • Ordering: Feeds display newest-first. The most recent event is always at the top of the stack.
