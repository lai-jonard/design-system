la-empty-state

The "nothing to show yet" surface for the legal area page. Two variants: processing renders immediately after unlock to say "we're working on it", and clean celebrates an area that has no open insights. The clean variant shows a Phosphor ph-check-circle icon by default, or a centred illustration image when the illustration prop is set.

Anatomy

Centred block with generous vertical padding. Variant decides whether the lead element is a spinning ring, a check-circle icon, or an illustration image.

la-empty-state

Legal area insights-column state surface. Two variants: processing (spinner, immediately after unlock) and clean (check-circle or illustration, when no insights are open). Display-only — no slots, no events.

Properties

PropertyAttributeTypeDefaultDescription
variant variant EmptyVariant 'clean' —
area area string '' —
message message string '' —
icon icon string '' Phosphor icon class for the neutral `default` variant, e.g. "ph ph-tray".
heading heading string '' Headline for the neutral `default` variant.
illustration illustration string '' URL of an illustration image. When set on the clean variant, renders a centred image in place of the check-circle icon.

Host padding uses --la-space-2xl --la-space-lg (32px / 16px). Illustration size is hardcoded at 120×120px — a design-level target size, no corresponding spacing token. Spinner geometry (24px square, 3px border) is also a fixed value. All three are accepted v1 exceptions.

Usage examples

Processing — immediately after unlock

<la-empty-state variant="processing" area="Governance" ></la-empty-state>

Sprint 3 decision — show processing immediately post-unlock, not a positive graphic. The user expectation is "analysis in progress", not "all done".

Clean — icon (default)

<la-empty-state variant="clean" area="Employment" ></la-empty-state>

Clean — with illustration

<la-empty-state variant="clean" area="Employment" illustration="/assets/illustrations/legal areas/employment_locked_intro.png" ></la-empty-state>

Drive variant from page state

const es = document.querySelector('la-empty-state'); es.variant = page.processing ? 'processing' : 'clean'; es.area = page.area; es.illustration = page.illustrationUrl;

Notes

Processing first, clean second. The legal area page renders variant="processing" immediately after unlock. variant="clean" is reserved for "analysis is complete and no issues found." Sprint 3 decision — see sprint-3-legal-area-page.md.
Illustration replaces the icon. When illustration is set, the ph-fill ph-check-circle icon is replaced by a 120×120 <img>. The illustration only applies to the clean variant — the processing variant always shows the spinner.
Centre-aligned by design. The host applies text-align: center. Drop it into any container with horizontal room and it lays out correctly without further wrapping.
Spinner is GPU-safe. The shared spin keyframe animates transform: rotate only. Respects prefers-reduced-motion via an in-shadow @media guard.
No retry path. The component states the situation but does not surface a retry button. If a step fails, the surrounding flow (e.g. la-unlock-panel failed state) owns retry.
