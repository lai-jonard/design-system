la-spinner

Indeterminate loading indicator — a small spinning ring used inside buttons, rows, and async-operation surfaces to communicate that work is in progress. Border colour inherits via currentColor; sizes follow the --la-spinner-* token scale.

Full rationale: design-language/loading.md §2.1–2.2. Family context: solves/260428-1-loading-states/.

Anatomy

An SVG with two concentric circles inside a <span role="status">. The host <la-spinner> sets display: inline-flex so it sits naturally inline with adjacent text or icons.

Anatomy — a single resting spinner
Internal structure
RegionWhat it is
Host <la-spinner> — sets display: inline-flex, forwards currentColor to the ring. appearance="invert" overrides the host colour to --la-color-text-invert for dark surfaces.
ARIA wrapper <span role="status" aria-label> — carries the live-region announcement so screen readers report the loading state.
Track A faint full-circle stroke at opacity: 0.2. Always visible, anchors the visual so the arc has something to spin against.
Arc The animated stroke. Two animations compose: the SVG rotates at 1.5s linear, while the arc length oscillates (5 → 70 → 5) at 1.4s ease-in-out. The result is a rotating segment that breathes.

Crossfade + scale (1 ⇄ 0.96), 200ms --ease-out. GPU-safe (transform + opacity only). Reduced-motion: instant swap. The trailing icon hides during loading rather than staying — keeps the loading state visually consistent regardless of original layout.

Width handling — preserved vs. not (both buttons content-sized; click each)
Width preserved (grid-stack — recommended) Sized to widest state ("Saving…"). No width jump on transition.
Width not preserved (instant content-swap — for comparison) Width changes mid-action; adjacent layout shifts. Avoid.

Both buttons are content-sized (no width set). The grid-stack pattern in the recommended version makes the button auto-size to whichever child is widest — so the loading state's "Saving…" determines the resting width, and the default state lives inside that width without growing or shrinking on transition. la-button[loading] will use this pattern internally so consumers don't have to think about it.

Usage examples

Inside la-button (anticipates the loading prop)

The spinner sits in the icon-left slot. Border inherits the button's foreground colour — white on filled variants, dark on ghost / secondary. This is the composition the future la-button[loading] prop will use internally.

<la-button variant="primary">
  <la-spinner slot="icon-left" size="sm" label="Saving"></la-spinner>
  Saving…
</la-button>

Inside la-upload-box

While a file is uploading, la-upload-box renders a la-spinner at size="md". The wrapping .zone sets color: var(--la-color-text-muted), so the spinner inherits the muted tint without an explicit colour declaration.

<la-spinner size="md" label="Uploading"></la-spinner>

Inside la-processing-widget

The processing-widget header tints its spinner via the --la-color-processing blue. The widget sets color on a wrapper rule; the spinner inherits.

<la-spinner size="sm" label="Processing"></la-spinner>

/* In the consuming component's CSS: */
.title la-spinner { color: var(--la-color-processing); }

Notes

currentColor inheritance

Both the track and arc strokes use currentColor. To tint, set color on the la-spinner host or any ancestor. The primitive does not expose a variant prop — colour is a property of the surrounding context, not of the spinner itself.

Dark backgrounds — appearance="invert"

For dark surfaces, set appearance="invert". The host colour switches to --la-color-text-invert (white), which both strokes inherit. Use this whenever the spinner sits on a dark backdrop and you don't want to manually set color on a wrapping element.

Reduced motion

The spinning animation respects prefers-reduced-motion automatically — when the OS setting is on, the ring becomes static. Behaviour lives in la-spinner.ts; nothing extra to wire up.

Family context

la-spinner is slot 1 of the loading-states family. The remaining slots are documented in foundations/loading — la-button[loading] (slot 2), la-skeleton (slot 3), la-thinking (slot 4), and la-progress (slot 5).
