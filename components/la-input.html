la-input

Text input with label, hint, icon slots, required, error, and disabled states. Wraps a native <input> element and fires a la-change event on every keystroke. Padding adjusts automatically when icon slots are populated.

Usage examples

Basic — no label

<la-input placeholder="Enter value" ></la-input>

Label + hint + required

<la-input label="Company name" placeholder="e.g. Acme Pty Ltd" hint="As registered with ASIC" required ></la-input>

Icon left — email

<la-input label="Email address" type="email" > <i slot="icon-left" class="ph ph-envelope"></i> </la-input>

Icon right — password reveal

<la-input type="password" id="pw"> <i slot="icon-right" class="ph ph-eye" style="cursor:pointer;pointer-events:all" ></i> </la-input>

Listen for changes

const input = document.querySelector('la-input'); input.addEventListener('la-change', e => { console.log(e.detail.value); });

Error state — validation

<la-input label="Email address" hint="Enter a valid email address" error required ></la-input>

Notes

Label association in shadow DOM — Each la-input generates a unique id for the inner <input> and links it to the <label> via for. Click the label — focus lands on the input.
Icon slot padding — Padding adjusts via CSS class toggling (.has-left / .has-right) driven by slotchange events. No boolean props needed — place an element in the slot and the component handles the rest.
Required asterisk — The * is aria-hidden="true". Screen readers rely on the required attribute forwarded to the inner <input>, not the visual asterisk.
Error and focus — When error is set, the focus glow (--la-input-shadow-focus) is suppressed so the red border reads clearly without a blue halo competing with it.
Name for forms — Set name when using inside a <form>. Forwarded to the inner input for native form submission. Note: web component form participation requires ElementInternals — use la-change for controlled forms.
Search — no label convention — Search inputs typically omit the label and rely on the icon and placeholder for context. Use type="search" with an icon-left slot and no label attribute.
Modal forms — la-input is designed for reuse in modals. Stack fields with gap: 16px inside a flex column. Pair with la-button for CTAs at the bottom of the form group.
