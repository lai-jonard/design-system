la-upload-box

A single-file drop zone primitive. Accepts click-to-browse and drag-and-drop. Used in the unlock flow's document upload steps and any form that needs a file. Wrapped by la-upload when source-buttons (Browse / Drive) and a transfer-error alert are needed; consumed directly when the drop zone alone is enough.

Notes

has-file state composes la-document-item. When the user selects a file, the zone's chrome no longer hand-rolls its own filename + remove-X. It renders an <la-document-item> with status="none" (no green check — the file has been selected, not yet processed), association="File selected", non-interactive, and dismissible. la-dismiss wires to the box's _remove. While processing, status flips to processing and association becomes "Uploading…". See decision 50.
has-file zone uses --la-shadow-card, not --la-shadow-sm. Plain white background + 1px border + soft three-layer card shadow. The border carries the edge, the shadow carries the elevation. See decision 51 for when to reach for this token vs --la-shadow-md / -float.
Error state has no separate retry button. The error treatment is inline (warning icon + message + hint) inside the zone, and the zone itself remains the click target — clicking anywhere re-opens the file picker, which clears the error on success. A separate "Choose another file" link inside the error zone would be redundant: cursor is already pointer, the dashed-border drop-zone affordance is intact.
Drag-over visual is two-channel. Border switches to --la-color-info + background tints to --la-color-processing-bg + text colour to --la-color-info-text. Border-only switching (dashed→solid same colour) was invisible to most users.
Drop multi-file rejects with reason 'multi'. The drop handler actively detects files.length > 1 and dispatches la-reject with reason; the picker still allows native multi-select but the box ignores all but the first.
Public method: openPicker(). Programmatically opens the file picker. Used by la-upload's "Browse files" button to drive the inner box without reaching into shadow DOM.
