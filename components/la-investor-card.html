la-investor-card

Per-investor summary card for the data room investor list. Shows the investor's name, firm, a three-state access pill (Not Shared / Active / Revoked), an optional last-activity line, and an unread-message count. The whole card is a single click target — selecting it fires la-investor-view so the host opens the per-investor detail. There is no per-card action menu; investor actions live on the detail surface (decision 70).

Anatomy

A single clickable card: the body (name, firm, meta row) on the left, and an optional unread badge on the right.

la-investor-card

Per-investor summary card for the data room — name, firm, three-state access pill, last activity, and unread count. Whole-card click opens the detail.

Properties

PropertyAttributeTypeDefaultDescription
investorName investor-name string '' Investor name, shown as the card title
firm firm string '' Firm name, shown below the title
status status InvestorStatus 'invited' Lifecycle state; drives the access pill and revoked dimming
lastActivity last-activity string '' Optional free-text activity summary
unreadCount unread-count number 0 Unread messages from this investor; shows a badge when > 0

Events

EventTypeDescription
la-investor-view CustomEvent Card clicked / activated; detail: { name }

CSS custom properties

No public CSS custom properties. All colour, spacing, radius, and shadow values resolve to design tokens — --la-color-bg, --la-color-border / -border-light, --la-radius-lg / -pill, --la-shadow-sm, --la-color-text / -text-muted / -text-faint, --la-color-primary / -primary-fg, and the focus-ring tokens.

Usage examples

Single card

<la-investor-card
  investor-name="James Okafor"
  firm="Meridian Capital"
  status="active"
  last-activity="Viewed 1 hour ago"
  unread-count="2"
  @la-investor-view=${e => openInvestor(e.detail.name)}
></la-investor-card>

List from a data array (Lit)

${investors.map(inv => html\`
  <la-investor-card
    .investorName=\${inv.name}
    .firm=\${inv.firm}
    .status=\${inv.status}
    .lastActivity=\${inv.lastActivity}
    .unreadCount=\${inv.unreadCount}
    @la-investor-view=\${() => select(inv)}
  ></la-investor-card>
\`)}

Notes

  • Whole-card click: the card is a native <button> with Enter / Space activation. There is no per-card action menu — investor actions (share, revoke, remove, nudge) live on the per-investor detail surface, per decision 70.
  • Three-state access pill: the seven lifecycle states collapse to Not Shared / Active / Revoked (Pattern 15). The pill is the card's status signal; access-revoked additionally dims the whole card to 60% opacity.
  • Progressive meta: last activity appears only when set; the unread badge appears only when unread-count > 0.
  • Soft motion divergence: the hover transition animates border-color and box-shadow (not the strict transform/opacity set) — the established hover-affordance pattern shared with la-legal-area-widget. A prefers-reduced-motion guard collapses it.
