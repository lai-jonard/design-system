Anatomy

The shell is a non-scrolling flex row: a fixed sidebar on the left and a content column on the right. The content column stacks the optional page header above a body row that splits into a scrollable main area and an optional aside panel.

slot: sidebar

la-sidebar
Full height. Never scrolls. Collapsible.

page header show-header

page-title · breadcrumbs · show-back
Never scrolls. Optional — many pages manage their own title.

slot: header-actions

Right-aligned. Buttons, widgets.

slot: main

Page content. The only independently scrolling region. overflow-y: auto.

slot: aside show-aside

Inline at desktop. Slide-in overlay at tablet (≤ 1023px content width).

Usage examples

Sidebar and content only

<la-app-shell style="height:100vh"> <la-sidebar slot="sidebar" ...></la-sidebar> <la-page slot="main"></la-page> </la-app-shell>

With page header and breadcrumbs

<la-app-shell show-header show-back> <la-sidebar slot="sidebar" ...></la-sidebar> <la-page slot="main"></la-page> </la-app-shell> // Set breadcrumbs via JS: shell.breadcrumbs = [ { label: 'Dashboard' }, { label: 'Governance' }, { label: 'Documents' }, ];

With aside panel and header actions

<la-app-shell show-header show-aside> <la-sidebar slot="sidebar" ...></la-sidebar> <la-button slot="header-actions">Export</la-button> <la-page slot="main"></la-page> <la-ai-chat slot="aside"></la-ai-chat> </la-app-shell>

Triggering aside from a child

// Any component inside the shell // can open the aside: element.dispatchEvent( new CustomEvent( 'la-toggle-aside', { bubbles: true, composed: true, } ) );

Notes

Height requirement — la-app-shell is display: flex; height: 100%. The parent must provide an explicit height — typically height: 100vh on the body or a full-height wrapper.
Container query, not viewport — The aside responsive breakpoint is a CSS container query on the content area (container-name: app-shell-content). The aside switches to overlay mode when the content area is ≤ 1023px wide, regardless of viewport size. A collapsed sidebar widens the content area and can shift the breakpoint without any JS.
Breadcrumb array — Set via JS property only (not an HTML attribute) since the value is an array: shell.breadcrumbs = [...]. When breadcrumbs is non-empty, it overrides pageTitle.
Aside overlay close — At tablet widths a close button (×) is automatically rendered at the top of the aside, and tapping the backdrop closes the panel. Both close via internal state — no event is emitted.
